A yet another blocky game.

## Running

Executable must be in the same folder as the `resources` folder.

## Binds

- Walking: `WASD`
- Jumping: `Space`
- Running: `Shift + W`
- Dash: `E` while in air
- Toggle V-Sync: `CTRL + V`
- Toggle debug information: `F3`
- Toggle mouse capture: `TAB`
- Select hotbar item: `1-9` or `Mouse Wheel`
